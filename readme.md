## FRONTEND ##

## TO RUN 

`npm run dev` 




## BACKEND ##

## creating a Virtual Environment

`python -m venv myenv`


## Installing Packages

`pip install -r requirements.txt`


## Running the Scripts

`myenv\Scripts\activate`


## Running the Backend

`python -m uvicorn app.main:app --reload`


## to exit from env 

`deactivate`
